# TestProject_WPF

You no longer need to create a new solution, When to create an test project.

If you need new test project, You are no longer Cumbersome to make new solution.
You jest download this project and build, Add new Window.xaml as much as if you want, in current project.
to ease as you know(Project - Add - New - Window.xaml)
It's just by adding up automatically display it (all of classes inheriting from "System.Windows.Window")
all of your window in the current project.

테스트 프로젝트를 생성하고 싶을 때, 이 솔루션을 사용하면 더 이상 새로운 솔루션을 만들지 않으셔도 됩니다.
프로젝트를 다운받아서, 빌드 후 이 프로젝트에서 (프로젝트 - 추가 - 새 항목 - Window.xaml)만
추가하셔도 자동적으로 당신의 Window.xaml을 실행시킬 수 있습니다.

리스트에는 현재 프로젝트에서 모든 System.Windows.Window를 상속하는 모든 클래스가 자동으로 들어가며,
리스트 클릭 후 시작 버튼만 누르면 간단하게 테스트 프로젝트를 생성할 수 있습니다!

이 프로젝트는 MIT 라이센스입니다 마음껏 All of xaml in the project사용하십시오!
